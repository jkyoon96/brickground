## 1. 설계 개요

### 1.1 설계 목표

| 목표 | 설명 |
|------|------|
| **이중 경험** | 사용자(Pixar)와 관리자(shadcn) 차별화된 UX |
| **몰입감** | 3D/VR 콘텐츠와 자연스럽게 연결되는 인터페이스 |
| **직관성** | 쇼핑/창작 과정의 인지 부하 최소화 |
| **일관성** | 통일된 디자인 언어와 패턴 |
| **접근성** | WCAG 2.1 AA 기준 준수 |

### 1.2 디자인 철학

```
┌─────────────────────────────────────────────────────────────────┐
│                     DESIGN PHILOSOPHY                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  사용자 UI: "Play & Create"                                      │
│  ───────────────────────────────────────────────────────────    │
│  장난스럽고 친근한 경험을 통해 쇼핑과 창작의 즐거움을 전달        │
│  마치 Pixar 영화 속에 들어온 듯한 따뜻하고 생동감 있는 공간       │
│                                                                  │
│  관리자 UI: "Efficient & Clear"                                  │
│  ───────────────────────────────────────────────────────────    │
│  깔끔하고 효율적인 인터페이스로 업무 생산성 극대화                │
│  데이터 중심의 명확한 정보 전달과 빠른 작업 처리                  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 1.3 UI 영역 구분

| 영역 | 대상 | 디자인 스타일 | URL 패턴 |
|------|------|--------------|----------|
| **User Frontend** | 일반 사용자, 창작자 | Pixar Style | `/`, `/products/*`, `/vrmalls/*`, `/classes/*` |
| **Seller Dashboard** | 판매자 | Pixar Style (Light) | `/seller/*` |
| **Admin Dashboard** | 관리자 | shadcn/ui | `/admin/*` |

### 10.5 prefers-reduced-motion 지원

```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

```typescript
// React Hook
const usePrefersReducedMotion = () => {
  const [prefersReducedMotion, setPrefersReducedMotion] = useState(false);

  useEffect(() => {
    const mediaQuery = window.matchMedia('(prefers-reduced-motion: reduce)');
    setPrefersReducedMotion(mediaQuery.matches);

    const handler = (e) => setPrefersReducedMotion(e.matches);
    mediaQuery.addEventListener('change', handler);
    return () => mediaQuery.removeEventListener('change', handler);
  }, []);

  return prefersReducedMotion;
};
```
