# BrickGround Product Backlog

> PRD 기반 Epic, Story, Task 분해  요구기능서 문서

## 넘버링 체계

| 구분 | 형식 | 설명 |
|------|------|------|
| Epic | `EP-XX` | 대분류 (기능 영역) |
| Story | `ST-XX-YY` | 중분류 (사용자 스토리) |
| Task | `TS-XX-YY-ZZ` | 소분류 (개발 작업) |

**우선순위**: P0 (필수) > P1 (중요) > P2 (선택)
